# CoAP Server 실행법

* node.coap.sample/server 디렉터리로 이동해주세요

* `npm install` 명령어를 통해 패키지를 설치해주세요

* 설치가 완료되면 `npm run start` 명령어 또는 `node main` 명령어를 통해 서버를 실행할 수 있습니다.

* 아래와 같은 화면이 나오면 서버가 정상적으로 작동하는 겁니다.
